# 接单管理系统 前端项目说明文档

## 项目结构
```tree
E:/2025/work/new/vite/order/order-ui/
├── public/                  # 静态资源目录
├── src/                     # 源代码目录
│   ├── assets/              # 静态资源
│   ├── components/          # 公共组件
│   ├── pages/               # 页面组件
│   ├── router/              # 路由配置
│   ├── store/               # 状态管理
```

## 使用说明

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产环境
```bash
npm run build
```
